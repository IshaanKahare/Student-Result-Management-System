# Student Result Management System

The Student Result Management System is a Python-based application that helps manage student academic records and calculate results efficiently. It allows users to add student details, enter subject marks, and automatically calculate total marks, percentage, and grade.

## Features
- Add student details
- Enter marks for different subjects
- Automatic calculation of total and percentage
- Grade calculation
- Display student results
- Simple graphical interface

## Technologies Used
- Python
- Tkinter (GUI)
